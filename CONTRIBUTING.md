# Contributing

Thank you for your interest in contributing.

## How to Contribute

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Submit a Pull Request

## Areas Where Help Is Needed

- Frontend/UI improvements
- Documentation
- Code refactoring
- Performance optimization
- Additional retrieval methods
- Testing

## Reporting Bugs

Please open an issue and include:

- Expected behavior
- Actual behavior
- Steps to reproduce

## Feature Requests

Feature requests are welcome. Please open an issue describing your idea and its potential benefits.
